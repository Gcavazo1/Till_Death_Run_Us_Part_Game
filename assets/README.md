# Game Assets

This directory contains all the visual and audio assets for "Til Death Run Us Part". All assets have been compressed for optimal performance.

## Visual Assets

### Player Character
- `zombie_bride_sheet.png` - Spritesheet with all zombie bride animations (running, turning left/right)

### Backgrounds
- `background.png` - Main graveyard background
- `iphone_background_landscape08.jpg` - iPhone frame for mobile display

### Obstacles
- `tombstone.png` - Main obstacle
- `spirit_orb.png` - Ghost-like obstacle

### Collectibles
- `pink_collectible.png`, `purple_collectible.png`, `red_collectible.png`, `green_collectible.png` - Basic collectible orbs
- `wedding_bouquet.png` - Special collectible (bonus points)
- `wedding_bands.png` - Special collectible (score multiplier)

### UI Elements
- `game_over.png` - Game over screen overlay

### Visual Effects
- `raven_left.png`, `raven_right.png` - Flying ravens
- `blue_eyes.png`, `red_eyes.png`, `green_eyes.png`, `yellow_eyes.png` - Peering eyes effect

## Audio Assets

### Music
- `spooky_scary_skeletons.mp3` - Main game music
- `spooky_time.mp3` - Alternative game music

### Sound Effects
- `start.mp3` - Game start sound
- `swipe.mp3` - Lane change sound
- `bouquet.mp3` - Wedding bouquet pickup
- `wedding_bands.mp3` - Wedding bands pickup
- `game_over.mp3` - Game over sound
- `collectible.mp3` - Regular collectible pickup
- `collide.mp3` - Obstacle collision

## Fonts
- `OctoberCrow.ttf` - Primary display font
- `Corpsy.otf` - Secondary display font

All assets have been properly optimized and compressed for web delivery.
